# 🎂 Birthday Template

A beautiful, interactive, and media-rich web template to celebrate birthdays with custom designs, animations, and music. Perfect for sending a personalized digital card to a friend or loved one.

👉 **[Live Preview Demo](https://ecxlipze.github.io/Birthday-Template/)**


## ✨ Features

- **🌸 Interactive UI & Micro-animations**: Soft gradients, rotating decorative items, and floating elements styled with modern CSS features.
- **🎵 Music Integration**: Includes built-in background music playback with play/pause controller controls.
- **🎀 Hello Kitty / Cute Aesthetics**: Styled with beautiful custom-crafted CSS components like a cute Hello Kitty bow and face decoration.
- **📱 Responsive Design**: Fully responsive layout that scales beautifully across mobile devices, tablets, and desktops.

## 🛠️ Built With

- **HTML5**: Core structure and content.
- **Vanilla CSS**: Styled with a customized HSL/Hex color palette, custom cursive typography using Google Fonts (Kalam), and interactive transition micro-animations.
- **Vanilla JavaScript**: Controls the music player state and dynamic interactivity.

## 🚀 How to Run Locally

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Ecxlipze/Birthday-Template.git
   cd Birthday-Template
   ```
2. **Open index.html**:
   - Simply double-click `index.html` to open it in your web browser.
   - Alternatively, serve it locally using any lightweight local server (e.g. Live Server extension in VS Code, or python server):
     ```bash
     python3 -m http.server 8000
     ```
     Then open `http://localhost:8000` in your browser.
